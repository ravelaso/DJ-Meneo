# dj-meneo
DJ Meneo - Discord Bot made with discordjs.

## How To Start

1. Install the dependencies:
  ```shell
  npm install
  ````
2. Create a .env file like the .env.example and fill the values.

3. Command to build:
  ```shell
  npm run build
  ```

4. Building for docker (requires Step: 3)
  ```shell
  docker build -t user/repo:tag .
  ```
